# React Counter App

This is a small counter application built using **React and Vite**.
The app allows the user to increase, decrease, and reset a counter value.

I created this project to practice **React components, state management, and project structure**.

---

## Features

* Increase the counter
* Decrease the counter
* Reset the counter to zero
* Simple component-based structure

---

## Project Structure

* **main.jsx** → Entry point of the React app
* **App.jsx** → Main component that renders the Counter component
* **Counter.jsx** → Contains the counter logic and buttons

---

## How to Run the Project

1. Install dependencies

```
npm install
```

2. Start the development server

```
npm run dev
```

3. Open the browser and go to

```
http://localhost:5173
```
